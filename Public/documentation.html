Flix It Up!

Developer Instructions

This is an API that will allow users to register an account, login, add movies to their favorites list, remove movies from their favorites list, view directors, view genres, and unregister their account.

Business Logic URL HTTP Method Request body data format Response body data format
Returns myFlix homepage / GET none A text message displaying homepage greeting
Returns myFlix Developer instructions /documentation GET none A text message describing all developer instructions
Allows new user to register myFlix account /users/register POST none A JSON object holding new user data. Example:
{ "Username": "newUser123",
"Password": "password123",
"Email": "newuser@email.com",
"Birthdate": "01/01/2000"
"Favorites": [ "Movie1", "Movie2"] //user has option to add movies during registration }
Returns a list of movies /movies GET none A JSON object holding data for all movies. Example:
{
id:1,
"Title": "Movie Title",
"Description": "Brief summary of movie",
"Genre": {
"Name": "Comedy",
"Description":"Brief definition of a comedy film"
}
"Director": {
"Name": "Director Adams",
"Bio": "Brief autobiography of director.",
"Birthdate": "01/01/1950",
"Birthplace": "Houston, Texas",
"Deathdate": "03/01/2010"
}
"ImagePath": "movieimage.png",
"Featured": ":true",
"Rating": "9/10",
"Release": "05/16/2016",
"Actors":[ Actor1, Actor2 ]
Allows user to update myFlix account /users/:Username PUT none A JSON object holding new data to be entered. Example:
{ "Username": "UpdateUser123",
"Password": "updatepassword123",
"Email": "updated@email.com",
"Birthdate": "01/01/2003"
, "Favorites": [ "Movie1", "Movie2"] }
Allows user to delete myFlix account /users/:Username/movies/:MovieID DELETE none A text message indicating user account was deleted
Allows user to filter movies by genre /movies/genres/:genres GET none A JSON object holding data for all movies with a specific genre
Allows user to search movies by title /movies/:titles GET none A JSON object holding data for a specific movie title
Allows users to add movies to a favorites list /users/:Username/movies/:MovieID POST A JSON object containing data about which movie to add to list A text message indicating movie was added to a user's account
Allows users to remove movies from a favorites list /users/:Username/movies/:MovieID DELETE none A text message indicating movie was removed from a user's account
Allows user to search movie directors /movies/directors/:names GET none A JSON object holding data for a specific movie director. Example:
"Director": {
"Name": "Director Adams",
"Bio": "Brief autobiography of director.",
"Birthdate": "01/01/1950",
"Birthplace": "Houston, Texas",
"Deathdate": "03/01/2010"
}
